# 전기조명설비 자동화 모듈

## 프로젝트 개요
이 프로젝트는 전기조명설비 자동화를 위한 임베디드 모듈을 개발하는 것을 목표로 했습니다. C 언어로 STM32 기반 제어 보드를 구현하고, LoRa E220-900T22D 모듈을 활용해 조명 상태와 센서 데이터를 수집하며 관제 서버의 명령에 따라 조명을 원격 제어할 수 있게 구성했습니다.

## 주요 기능
- PIR 센서를 이용한 인체 감지 정보 수집
- 전등 ON/OFF 상태 정보 수집
- CDS 조도 센서를 이용한 조명 환경 정보 수집
- 관제 시스템으로부터 전등 전체 ON/OFF 명령 수신
- 전등 꺼지는 시간 설정 명령 수신 및 실행
- 전등 고장 상태 확인 기능 지원

## 개발 과정에서의 문제와 해결
1. PIR 센서 오감지
   - PIR 센서의 가변저항 조정만으로는 오감지가 줄지 않아 불필요한 조명 점등이 발생했습니다.
   - 원인을 분석한 결과, 패킷 송수신 시 PIR 센서가 튀는 현상이 있었고 소프트웨어적으로 송수신 구간 동안 감지를 차단해 문제를 해결했습니다.
2. LoRa 통신 패킷 누락
   - 다수의 모듈이 동시에 통신하는 환경에서 송신 중 수신 패킷이 누락되는 문제가 발생했습니다.
   - 누락된 모듈을 식별하고 재전송하는 로직을 추가해 통신 안정성을 향상시켰습니다.

## 학습 및 성장
이 프로젝트는 임베디드 소프트웨어 현업에서 처음으로 진행한 프로젝트였고, 문제 발생 지점을 스스로 점검하는 능력을 키울 수 있었습니다. 멀티미터와 오실로스코프를 활용해 전압, 전류, 센서 신호를 확인하는 경험은 이후 다양한 문제 상황에서도 원인을 빠르게 파악하는 데 큰 도움이 되었습니다.
